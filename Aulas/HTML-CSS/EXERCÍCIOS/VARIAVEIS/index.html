Porsche Carrera GT

O Carrera GT foi lançado comercialmente em 2004, mas o conceito de mesmo nome surgiu em 2000. A Porsche aproveitou a data original para comemorar os 20 anos do modelo. O superesportivo surgiu como derivado de um projeto para Le Mans, um protótipo nunca executado que seria o substituto natural do 911 GT1. Para não desperdiçar os belos esforços, a marca alemã o lançou como carro de rua. Fez muito bem. O carro estilo targa criou uma mística própria. Foram construídos 1.270 exemplares. Até hoje sua ficha técnica é capaz de causar comoção em entusiastas. O motor V10 tem 5,7 litros e fornece 612 cv a divinos 8.000 rpm. Grande parte do torque de 60,2 kgfm está disponível em baixas rotações, enquanto a força máxima é entregue a 5.500 giros.
